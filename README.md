### **Getting Started**  

#### **1. Clone the Repository**
```sh
git clone https://github.com/yourusername/planning-poker.git
cd planning-poker
```

#### **2. Setup Backend**
```sh
cd backend
npm install
npm start
```

#### **3. Setup Frontend**
```sh
cd ../frontend
npx create-react-app .
npm install
npm start
```

#### **4. Open in Browser**
Go to: [`http://localhost:3000`](http://localhost:3000)
